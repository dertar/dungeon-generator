### dungeon-generator ###

the main tricks of rooms generate are taken from [spelunky](http://tinysubversions.com/spelunkyGen/)

controls:
key - descriptions
0 - new level
9 - show room marks
8 - show box2d
7 - toggle camera
6 - show userdata for correct collision
?5 - switch light

q - zoom in
a - zoom out
w - rotation by clock
e - rotation by !clock 

z - jump
arrows - control
